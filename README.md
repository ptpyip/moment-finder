# Moment Finder

> Retrieve moments from videos using natural language query



## Requirments
```bash
conda env create -n <name> -f environment.yml
conda activate <name>
pip install -r ./requirements.txt
pip install git+https://github.com/openai/CLIP.git
```

## Start API
```bash
python -m app.api
```
